Automated Hindi Headline Generation

Designed and developed a Python head line generator for a given news article in Hindi. The algorithm was trained on 1000 news articles and used concepts based on content selection features like Parts of speech, Bi-gram and trigram of current story word. This generator was based on algorithms like POS Language Model Feature (based on POS trigrams), Headline Length Feature, Content Selection Feature and BLEU scores.

Video Search Application

Developed a JAVA based search like application to compare and calculate similarity in the given set of videos. The HSB values (Hue, Saturation, Brightness) of pixels for each frame (divided into a set of blocks), were compared to calculate the percentage match with database videos. The final output printed the similarity of the input video with the set of database videos in descending order of percentage match.

eBay Search - Web and Android applications
